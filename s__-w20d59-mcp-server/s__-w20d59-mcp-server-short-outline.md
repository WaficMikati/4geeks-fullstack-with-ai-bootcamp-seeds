## 💻 Declaration
YES — Students build a Python MCP Server using the Python MCP SDK that exposes at least one functional tool (e.g., a data lookup or calculation tool). The server runs and produces observable output when an MCP client connects and calls the tool.

**Note:** "Introducción a MCPs" covered what MCPs are, architecture, and client/server roles. This learnpack builds directly on that — no re-introduction of MCP concepts. MCP Client implementation (in TypeScript) is the next learnpack and is not covered here.
