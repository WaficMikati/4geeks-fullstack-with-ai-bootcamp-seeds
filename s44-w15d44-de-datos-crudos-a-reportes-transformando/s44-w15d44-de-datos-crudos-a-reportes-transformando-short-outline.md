# Course Outline: From Raw Data to Reports

**Skill:** Skill 44: Building a report from the telemetry or user data

💻 **Declaration:** YES — students write Python code using pandas to load raw TODO data, filter by status and date, group by day, aggregate with count, and output the result as JSON. The code runs and produces observable output (a grouped/aggregated DataFrame printed or served as JSON).
