**💻 Declaration:** YES — Students write Next.js code at three capture points: a router listener (routeChangeComplete) that fires page view events, an event handler attached to a key user action (submit/enroll/search), and a global error handler (window.onerror or Error Boundary) and/or a fetch interceptor. All produce observable output — telemetry payloads sent to the endpoint built in the previous learnpack.
