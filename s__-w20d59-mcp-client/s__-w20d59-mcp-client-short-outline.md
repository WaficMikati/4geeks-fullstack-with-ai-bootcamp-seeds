## 💻 Declaration
YES — Students write TypeScript code to build an MCP Client that connects to an MCP Server, lists its available tools, and calls one of them with arguments. The client produces observable output: tool response data printed to the console.

**Note:** MCP architecture and client/server roles were covered in "Introducción a MCPs". MCP Server (Python) was covered in the preceding learnpack. This learnpack is the consumer side, using TypeScript and the MCP TypeScript SDK.
