## 💻 Declaration
YES — Students build a Python CLI that returns structured JSON output, designed to be called programmatically by an AI agent. The tool accepts arguments and produces machine-readable responses (JSON) and exit codes that an agent can parse and act on.

**Note:** Immediately preceding learnpack ("CLIs") covered what CLIs are, why humans love them, and how to build one with Click/argparse. This learnpack builds on that foundation and focuses on what changes when the consumer is an AI agent instead of a human. MCPs (in the same skill block) are reserved for the next learnpack — MCP concepts are not introduced here.
